# WE MZEE Investment Group Management System

A comprehensive web application for managing investment group operations, including contributions, investments, loans, and profit distribution.

## Features

### For Members
- **Dashboard**: Overview of financial activities
- **Contributions**: Submit and track monthly contributions
- **Investments**: View investment portfolio and performance
- **Loans**: Apply for loans and track repayments
- **Profits**: View profit distributions and history
- **Profile**: Manage personal information

### For Administrators
- **Member Management**: View and manage all members
- **Reports**: Generate financial reports and statements
- **Notifications**: Send broadcast messages and reminders
- **Approval System**: Approve contributions and loans

## Tech Stack

### Frontend
- React 18
- React Router DOM
- Tailwind CSS
- Chart.js for data visualization
- Lucide React for icons
- Axios for API calls

### Backend
- Node.js with Express
- MongoDB with Mongoose
- JWT for authentication
- PDFKit for report generation
- Nodemailer for email notifications
- Twilio for SMS (optional)
